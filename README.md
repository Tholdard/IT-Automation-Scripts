# IT-Automation-Scripts

A collection of beginner-friendly Python and Bash scripts for common IT automation, system administration, and network troubleshooting tasks.

## Overview
This repository was created to practice building practical scripts that can assist with system information gathering, disk usage monitoring, network testing, file backups, and log analysis. The goal is to develop hands-on scripting experience relevant to IT support, systems administration, and technical troubleshooting roles.

## Technologies Used
- Python 3
- Bash
- Git
- GitHub
- Visual Studio Code
- Git Bash / Linux terminal

## Python Scripts

### 1. System Info Script
**Path:** `python/system_info/system_info.py`

Displays basic system information such as:
- hostname
- operating system
- OS version
- machine type
- processor
- IP address
- Python version
- current user

**How to run:**
```bash
python python/system_info/system_info.py
